# fat-fingers

> Slightly clumsy, mostly useful Claude plugins.

A small marketplace of [Claude Code](https://code.claude.com) plugins.

## Install

Add the marketplace, then install the plugin you want:

```bash
/plugin marketplace add t3spe/fat-fingers
/plugin install humanize-it@fat-fingers
```

Update later with:

```bash
/plugin marketplace update fat-fingers
/plugin update --all
```

## Plugins

| Plugin | What it does |
|---|---|
| [`humanize-it`](./plugins/humanize-it) | Rewrite AI-sounding text so it reads like a human wrote it. |
